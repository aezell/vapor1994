# Vapor1994

Vapor1994 is a two-column theme for micro.blog sites.

## Contents

- [Usage](#usage)
- [Author](#author)
- [License](#license)

## Usage

To install Vapor1994 on your micro.blog site, just use this repository's URL in the Design section of your site.
